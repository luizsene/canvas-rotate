canvasRotate
directive in module br.com

Description

Diretiva que fornece a possibilidade de rotacionar imagens para edição

Dependencies

  • $ionicGesture

Usage

as attribute
<any canvas-rotate>
   ...
</any>

Directive info

  • This directive creates new scope.
